**Проблема:** отсутствие централизованной базы данных, долгое время выполнения ad-hoc задач.

**Задача:** создать базу данных, к которой можно обращаться при выполнении разовых аналитических задач. База должна быть локальная.

Инструменты: DBeaver

**Ключевые этапы:**

1. Написание скриптов для предобработки данных перед загрузкой в базу.
2. Создание таблиц и ключей.
3. Создание индексов.
4. Заполнение базы.

**Результат:** работающая база, которая сократила время выполнения аналитических ad hoc задач до десятков минут. Денежные затраты - 0.
