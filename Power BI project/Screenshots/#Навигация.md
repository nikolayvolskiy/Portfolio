**Основная информация – 1:**

-	Сводка за месяц: Общая выручка, возвраты, доля возвратов.
-	Эффективность точек продаж: Сравнение точек по ключевым метрикам: количество клиентов (новых/старых), выручка, возвраты.
-	Распределение продаж: Доли точек в общей чистой выручке (после возвратов).
-	Детализация по точкам: Продажи в рублях и штуках, количество чеков, клиентов, средний чек, средняя цена, среднее количество товаров в чеке (средняя корзина).

**Основная информация - 2:**

-	Временные ряды (по неделям): Динамика ключевых показателей.
-	Графики: Продажи в рублях, продажи в единицах, средний чек, количество клиентов (новых и всего), количество чеков, средняя корзина.

**Основная информация - 3:**

-	ABC-анализ по товарным категориям: Ранжирование категорий по вкладу в выручку. 

**Основная информация - 4:**

-	Топ товаров по доходности (рубли): Список наиболее доходных артикулов (SKU) за период с указанием проданного количества и выручки.

**Основная информация - 5:**

-	Распределение товаров по ключевой характеристики – сезонности.

**Основная информация - 6:**

- Анализ трафика и конверсии (по неделям): Таблица и график с еженедельным трафиком, количеством чеков и расчётным показателем конверсии (Чеки/Трафик).

**Для продуктовой команды - 1:**

- Продажи по всем категориям: Полный список всех товарных категорий с показателями выручки и количества проданных единиц за период.
- Гистограмма: Визуализация рейтинга категорий по общей выручке.

**Для продуктовой команды - 2:**

- Топ товаров (артикулов) по выручке: Детализированный список самых доходных SKU за период с указанием названия, количества проданных единиц и суммы выручки.

**Для продуктовой команды - 3:**
- Анализ ассортимента по конкретным товарам: Срез данных по выбранным товарам.
- Распределение продаж по атрибутам: цвет и размер.
- По цветам: показывает популярность отдельных цветов в количестве проданных единиц.

**Для топ менеджмента - 1:**

Основные операционные метрики в динамике: набор графиков, показывающих месячные тренды.
Анализируемые показатели: Выручка (руб.), объём продаж (шт.), количество чеков (транзакций), общее количество клиентов, а также отдельно динамика по старым и новым клиентам.

**Для топ менеджмента - 2:**

Метрики эффективности и покупательского поведения в динамике: Продолжение анализа месячных трендов за тот же период.

Анализируемые показатели: Средний чек, среднее количество товаров в чеке (средняя корзина), трафик и количество чеков на одном графике для сравнения, конверсия (в %), средняя цена товара.

**Детализация по сотрудникам - 1:**

Рейтинг сотрудников точки продаж: Сравнение показателей работы отдельных продавцов (сотрудников) в выбранном магазине за период.

Ключевые метрики на сотрудника: Выручка, количество проданных единиц, количество чеков, средняя корзина, средний чек, средняя цена.

Визуализация: Гистограмма, показывающая вклад каждого сотрудника в общую выручку точки.

**Детализация по сотрудникам - 2:**

Динамика основных показателей сотрудника: Набор графиков, показывающих изменение ключевых метрик выбранного сотрудника в помесячной разбивке за период.

Анализируемые показатели: Выручка, средняя корзина, количество чеков, средняя цена, средний чек.

**Детализация по сотрудникам - 3:**

Вклад категорий в продажи сотрудника: Рейтинг товарных категорий по выручке и количеству проданных единиц в продажах выбранного сотрудника за период.

Визуализация: Гистограмма, показывающая распределение выручки по категориям.

**Детализация по сотрудникам - 4:**

Разбор выбранной категории между сотрудниками: Вклад каждого сотрудника в продажи определенной категории.

Визуализация: Гистограмма, показывающая распределение выручки по сотрудникам.

**Детализация по сотрудникам - 5:**

Топ товаров (артикулов) сотрудника: Список самых доходных SKU у выбранного сотрудника за период с указанием выручки и проданного количества.

**Детализация для b2b команды - 1:**

Рейтинг клиентов (B2B): Сравнение ключевых клиентов (компаний) по объёму закупок за период.

Метрики на клиента: Количество закупленных единиц, общая сумма выручки от клиента, средняя цена закупаемого товара.

Визуализация: Три гистограммы, показывающие рейтинг клиентов по выручке, по количеству закупленных единиц и по средней цене закупаемого товара.

**Детализация для b2b команды - 2:**

Структура продаж по категориям для B2B: Список всех товарных категорий с показателями выручки и количества проданных единиц в рамках B2B-направления за период.

Визуализация: Гистограмма, показывающая распределение B2B-выручки по категориям.

**Детализация для b2b команды - 3:**

Детализация B2B-продаж по товарам (артикулам): Список товаров (SKU), проданных в рамках B2B-направления, с указанием выручки и проданного количества.

**Детализация по клиентам - 1**:

Структура продаж по типу клиента (старый/новый): Сравнение ключевых показателей.

Визуализация: Гистограммы, показывающие разницу в показателях в разных групп клиентов.

**Детализация по клиентам - 2**:

Помесячная динамика ключевых показателей клиентов.

Визуализация: Линейный графики.

**Детализация по клиентам - 3**:

Структура продаж по категориям в разрезе типа клиента: Список всех товарных категорий с показателями выручки и количества проданных единиц либо для старых клиентов, либо для новых.

Визуализация: Гистограмма, показывающая распредление выручки.

**Детализация по клиентам - 4.:**

Анализ продаж по сезонам: Итоги по выбранным категориям товаров, сгруппированные по сезонным коллекциям.

Топ товаров: Список самых доходных артикулов в рамках выбранных категорий за период.

Распределение по сезонам: Сравнительная таблица и круговая диаграмма, показывающие вклад каждой сезонной коллекции (Сезон 1, Сезон 2 и т.д.) в общую выручку и количество проданных единиц.

**Детализация маркетинговым показателям - 1:**

Когортный анализ. Retention rate (Удержание): Матрица, показывающая процент клиентов из каждой когорты (по кварталам), совершивших повторную покупку в последующие периоды. Анализ "сохранности" клиентской базы.

**Детализация маркетинговым показателям - 2:**

Когортный анализ. LTV (Lifetime Value): Матрица, показывающая совокупную выручку (нарастающим итогом), полученную от каждой когорты клиентов за весь период её жизни.

**Детализация по маркетинговым показателям - 3:**

Визуализация динамики LTV по когортам: Линейный график, на котором отображены кривые роста Lifetime Value для разных клиентских когорт во времени. Позволяет сравнивать долгосрочную ценность клиентов, пришедших в разные периоды.

##
На дашбордаж можно настроить фильтр по дате (выбрать произвольный временной период), по направление и по точке продаж. На дашбордах с более детальной информацией можно так же настроить фильтр на категорию товара и сезон, выбрать сотрудника или b2b клиента, выбрать тип клиента (старый или новый). В дашборде с маркетинговыми показателями можно фильтровать только когорты привлечения (квартальные) и когорту покупки (так же квартальную).
