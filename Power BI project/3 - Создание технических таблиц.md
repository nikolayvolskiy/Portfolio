В дополнение к имеющимся таблицам создаем дополнительные таблицы, которые будут служить для обеспечения расчетов и возможности фильтрации по временному интервалу.

**Таблица calendar** - создается обеспечения связи с датами в исходных таблицах для возможности фильтрации данных по дате.

```dax
ADDCOLUMNS (
    CALENDAR (DATE(2020,06,1), DATE(2026,12,31)),
    "year", YEAR([Date]),
    "month", MONTH([Date]),
    "month_name", FORMAT([Date], "MMMM"),
    "quarter", "Q" & QUARTER([Date]),
    "day_of_week", WEEKDAY([Date]),
    "day_name", FORMAT([Date], "DDDD"),
    "week_number", WEEKNUM([Date], 2),
    "year_month", FORMAT([Date], "YYYY-MM"),
    "year_week", FORMAT([Date], "YYYY") & "-W" & FORMAT(WEEKNUM([Date], 2), "00")
)
```

**Таблица all_sales** - вспомогательная таблица для определения новых покупателей.

```dax
all_sales = 
UNION(
    SELECTCOLUMNS(
        main_sales, 
        "customer_phone_fixed", main_sales[customer_phone_fixed],
        "sales_date", main_sales[sales_date]
    ),
    SELECTCOLUMNS(
        com_sales, 
        "customer_phone_fixed", com_sales[customer_phone_fixed],
        "sales_date", com_sales[sales_date]
    ),
    SELECTCOLUMNS(
        e_com_sales, 
        "customer_phone_fixed", e_com_sales[customer_phone_fixed],
        "sales_date", e_com_sales[sales_date]
    )
)
```

**Таблица cohort_analysis_base** – таблица собирает все продажи и делает столбцы с датой первой покупки для конкретного покупателя. Данная таблица будет служить для анализа показателя возвращаемости (Retention Rate).

```dax
cohort_analysis_base = 
SUMMARIZE(
    ADDCOLUMNS(
        all_sales_2,
        "sales_quarter", FORMAT([sales_date], "YYYY") & "-Q" & QUARTER([sales_date]),
        "first_purchase_quarter", 
            CALCULATE(
                FORMAT(MIN(all_sales_2[sales_date]), "YYYY") & "-Q" & QUARTER(MIN(all_sales_2[sales_date])),
                ALLEXCEPT(all_sales_2, all_sales_2[customer_phone_fixed])
            )
    ),
    [customer_phone_fixed],
    [first_purchase_quarter],
    [sales_quarter],
    "is_active", 1
)
```

**Таблица net_sales** – собранные вместе продажи и возвраты (со знаком минус). Служит вспомогательной таблицей для подсчета показателя пожизненной ценности покупателя (LTV).

```dax
net_sales = 
VAR sales = 
    UNION(
        SELECTCOLUMNS(main_sales, "customer", [customer_phone_fixed], "date", [sales_date], "value", [total_value]),
        SELECTCOLUMNS(com_sales, "customer", [customer_phone_fixed], "date", [sales_date], "value", [total_value]),
        SELECTCOLUMNS(e_com_sales, "customer", [customer_phone_fixed], "date", [sales_date], "value", [total_value])
    )
VAR returns = 
    UNION(
        SELECTCOLUMNS(main_returns, "customer", [customer_phone_fixed], "date", [return_time], "value", -[total_value]),
        SELECTCOLUMNS(com_returns, "customer", [customer_phone_fixed], "date", [return_time], "value", -[total_value])
    )
RETURN
UNION(sales, returns)
```

Таблица ltv_analysis – собирает данные из вспомогательной таблицы net_sales и присваивает квартал первой и текущей покупок для возможности проведения когортного анализа.

```dax
ltv_analysis = 
ADDCOLUMNS(
    SUMMARIZE(
        net_sales,
        net_sales[customer],
        net_sales[date]
    ),
    "cohort", FORMAT(CALCULATE(MIN(net_sales[date]), ALLEXCEPT(net_sales, net_sales[customer])), "YYYY") & "-Q" & QUARTER(CALCULATE(MIN(net_sales[date]), ALLEXCEPT(net_sales, net_sales[customer]))),
    "quarter", FORMAT(net_sales[date], "YYYY") & "-Q" & QUARTER(net_sales[date]),
    "revenue", CALCULATE(SUM(net_sales[value]), ALLEXCEPT(net_sales, net_sales[customer], net_sales[date]))
)
```

После добавления таблицы all_sales в основные таблицы с продажами - main_sales, com_sales и e_com_sales нужно добавить столбец определения новой покупки.

```dax
is_first_purchase = 
VAR CurrentCustomer = main_sales[customer_phone_fixed]
VAR CurrentDate = main_sales[sales_date]
VAR FirstPurchaseDate = 
    IF(
        NOT ISBLANK(CurrentCustomer),
        CALCULATE(
            MIN(all_sales[sales_date]),
            FILTER(
                ALL(all_sales),
                all_sales[customer_phone_fixed] = CurrentCustomer &&
                all_sales[customer_phone_fixed] <> BLANK()
            )
        ),
        BLANK()
    )
RETURN
IF(
    ISBLANK(CurrentCustomer),
    BLANK(),
    IF(
        CurrentDate = FirstPurchaseDate, 
        "Новый клиент", 
        "Старый клиент"
    )
)
```
