**СВЯЗИ С CALENDAR (Date → Date):**

calendar[Date] → main_sales[sales_date] (один ко многим, активная)  
calendar[Date] → com_sales[sales_date] (один ко многим, активная)  
calendar[Date] → e_com_sales[sales_date] (один ко многим, активная)  
calendar[Date] → b2b_sales[sales_date] (один ко многим, активная)  
calendar[Date] → main_returns[return_date] (один ко многим, активная)  
calendar[Date] → com_returns[return_date] (один ко многим, активная)  
calendar[Date] → e_com_returns[return_date] (один ко многим, активная)  
calendar[Date] → traffic_butik[traffic_date] (один ко многим, активная)  
calendar[Date] → all_sales_2[sales_date] (один ко многим, активная)
