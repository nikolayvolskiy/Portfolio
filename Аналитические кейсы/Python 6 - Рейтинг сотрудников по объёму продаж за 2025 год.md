## Рейтинг сотрудников по объёму продаж за 2025 год

**Цель анализа:**

Составить ранжированный список сотрудников по ключевым показателям продаж за 2025 год, выявить:

- лучших продавцов по сумме выручки;

- лидеров по количеству проданных товаров;

- вклад каждого сотрудника в общий доход компании.

**Почему это важно:**

- позволяет объективно оценить эффективность работы персонала;

- помогает формировать систему мотивации и премий на основе реальных результатов;

- выявляет сотрудников, нуждающихся в дополнительном обучении;

- даёт основу для планирования KPI и развития команды продаж.

Выведите рейтинг сотрдуников по выручке за 2025 год. В качестве визуализации представьте линейчатую диаграмму
```python
import pandas as pd
import matplotlib.pyplot as plt

#Загрузка данных
df_sales = pd.read_csv('C:/Python/t1_main_sales.csv', sep=';', encoding='cp1251', low_memory=False)
df_employee = pd.read_csv('C:/Python/t1_employees.csv', sep=';', encoding='cp1251', low_memory=False)

#Преобразование даты и фильтрация за 2025 год
df_sales['sales_time'] = pd.to_datetime(df_sales['sales_time'], errors='coerce')
sales_2025 = df_sales[df_sales['sales_time'].dt.year == 2025]

#Группировка по employee_id и подсчет метрик
employee_sales = sales_2025.groupby('employee_id').agg(
    total_qty=('qty', 'sum'),
    total_value=('total_value', 'sum')
).reset_index()

#Добавление имен сотрудников
employee_sales = employee_sales.merge(df_employee[['employee_id', 'employee_name']], on='employee_id', how='left')

#Рассчитываем общую выручку и процент для каждого сотрудника
total_revenue = employee_sales['total_value'].sum()
employee_sales['revenue_percent'] = (employee_sales['total_value'] / total_revenue * 100).round(2)

#Сортировка по возрастанию total_value
employee_sales = employee_sales.sort_values(by='total_value', ascending=True)

#Вывод таблицы
result_table = employee_sales[['employee_name', 'total_qty', 'total_value', 'revenue_percent']].copy()
result_table = result_table.rename(columns={
    'employee_name': 'Имя сотрудника',
    'total_qty': 'Количество проданных товаров',
    'total_value': 'Сумма выручки',
    'revenue_percent': 'Доля от общей выручки, %'
})

print("Рейтинг сотрудников по продажам за 2025 год:")
print(result_table.to_string(index=False))

#Построение линейчатой диаграммы
plt.figure(figsize=(12, 8))

#Создаем горизонтальную диаграмму
bars = plt.barh(employee_sales['employee_name'], employee_sales['total_value'], color='lightcoral', height=0.7)

#Добавляем значения на диаграмму
for bar, value, percent in zip(bars, employee_sales['total_value'], employee_sales['revenue_percent']):
    width = bar.get_width()
    plt.text(width + (total_revenue * 0.005), 
             bar.get_y() + bar.get_height()/2, 
             f"{width:,.0f} руб.\n({percent}%)", 
             va='center', 
             fontsize=9)

plt.xlabel('Сумма выручки, руб.')
plt.ylabel('Сотрудник')
plt.title('Рейтинг сотрудников по выручке за 2025 год')
plt.grid(axis='x', linestyle='--', alpha=0.7)
plt.xlim(0, total_revenue * 1.15)
plt.tight_layout()
plt.show()
```

**Что делает код:**

Загрузка данных:

- df_sales — транзакции продаж (объёмы, суммы, даты, ID сотрудника);

- df_employee — справочник сотрудников (ID, ФИО).

Подготовка дат:

- преобразует столбец sales_time в формат datetime;

- фильтрует записи только за 2025 год.

Агрегация по сотрудникам:

- группирует продажи по employee_id;

  суммирует:

  - total_qty — количество проданных товаров;

  - total_value — выручка в рублях.

Связывание с именами сотрудников:

- присоединяет колонку employee_name через объединение с df_employee.

Расчёт долей:

- вычисляет общую выручку за период (total_revenue);

- определяет долю каждого сотрудника в общей выручке (revenue_percent).

Сортировка и форматирование:

- упорядочивает сотрудников по возрастанию выручки (ascending=True);

- формирует итоговую таблицу с понятными заголовками.

Визуализация:

- строит горизонтальную линейчатую диаграмму (barh);

- для каждого сотрудника отображает:

- длину столбца — сумма выручки;

- подпись — выручка в рублях и доля в %;

- настраивает сетку, подписи осей и заголовок.

Как читать результат:

Таблица:

строки — сотрудники в порядке возрастания выручки;

колонки:

Имя сотрудника — ФИО продавца;

- Количество проданных товаров — объём продаж в штуках;

- Сумма выручки — итоговая выручка в рублях;

- Доля от общей выручки, % — вклад сотрудника в совокупный доход.

Диаграмма:

- ось Y — имена сотрудников (снизу вверх: от минимальной к максимальной выручке);

- ось X — сумма выручки в рублях;

- длина столбца — величина выручки;

- подпись у конца столбца — точная сумма и процент от общего объёма.

Примеры интерпретации:

Длинные столбцы справа — лучшие продавцы. Стоит:

- изучить их техники работы с клиентами;

- рассмотреть возможность наставничества для менее успешных коллег;

- предусмотреть бонусы и поощрения.

- Короткие столбцы слева — сотрудники с низкой выручкой. Требуется:

  - анализ причин (недостаток навыков, сложная локация, внешние факторы);

  - дополнительное обучение;

  - пересмотр зоны ответственности.

Высокая доля нескольких сотрудников — риск зависимости от ключевых продавцов. Рекомендуется:

- развивать командные продажи;

- внедрять кросс‑продажи;

- распределять клиентскую базу равномерно.

Равномерное распределение долей — сбалансированная команда, где каждый вносит сопоставимый вклад.

Резкий разрыв между лидерами и аутсайдерами — сигнал к:

- ревизии системы обучения;

- корректировке KPI;

- анализу загрузки сотрудников.

Ключевые выводы:

Топ‑3 продавца — основные драйверы дохода. Их опыт ценен для тиражирования.

Сотрудники в нижней трети рейтинга — кандидаты на дополнительное обучение или ротацию.

Доля лидера в общей выручке показывает степень концентрации продаж на отдельных сотрудниках.

Соотношение количества товаров и суммы выручки выявляет продавцов с высоким средним чеком.
