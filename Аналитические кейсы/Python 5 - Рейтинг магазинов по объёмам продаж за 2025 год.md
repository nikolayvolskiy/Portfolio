## Рейтинг магазинов по объёмам продаж за 2025 год

**Цель анализа:**

Составить ранжированный список торговых точек по ключевым показателям продаж за 2025 год, выявить:

 - магазины‑лидеры и аутсайдеры по выручке;

- распределение доли рынка между точками;

- объёмы реализованной продукции в натуральном выражении.

**Почему это важно:**

- позволяет оценить эффективность работы каждого магазина;

- помогает распределить ресурсы (персонал, закупки, маркетинг) в пользу наиболее результативных точек;

- выявляет потенциальные проблемы в отстающих магазинах;

- даёт основу для постановки целевых показателей и мотивации персонала.

Определить рейтинг магазинов по продажам за 2025 год. Таблица должна содержать store_name, количество проданных товаров, сумму выручки и отсортирована по возрастанию. В качестве визуализации сформируйте линейчатую диаграмму.

```python
import pandas as pd
import matplotlib.pyplot as plt

#Загрузка данных
df_sales = pd.read_csv('C:/Python/t1_main_sales.csv', sep=';', encoding='cp1251', low_memory=False)
df_stores = pd.read_csv('C:/Python/t1_stores.csv', sep=';', encoding='cp1251', low_memory=False)

#Преобразование даты и фильтрация за 2025 год
df_sales['sales_time'] = pd.to_datetime(df_sales['sales_time'], errors='coerce')
sales_2025 = df_sales[df_sales['sales_time'].dt.year == 2025]

#Группировка по store_id и подсчет метрик
store_sales = sales_2025.groupby('store_id').agg(
    total_qty=('qty', 'sum'),
    total_value=('total_value', 'sum')
).reset_index()

#Добавление названий магазинов
store_sales = store_sales.merge(df_stores[['store_id', 'store_name']], on='store_id', how='left')

#Рассчитываем общую выручку и процент для каждого магазина
total_revenue = store_sales['total_value'].sum()
store_sales['revenue_percent'] = (store_sales['total_value'] / total_revenue * 100).round(2)

#Сортировка по возрастанию total_value
store_sales = store_sales.sort_values(by='total_value', ascending=True)

#Вывод таблицы
result_table = store_sales[['store_name', 'total_qty', 'total_value', 'revenue_percent']].copy()
result_table = result_table.rename(columns={
    'store_name': 'Название магазина',
    'total_qty': 'Количество проданных товаров',
    'total_value': 'Сумма выручки',
    'revenue_percent': 'Доля от общей выручки, %'
})

print("Рейтинг магазинов по продажам за 2025 год:")
print(result_table.to_string(index=False))

#Построение линейчатой диаграммы
plt.figure(figsize=(12, 8))

#Создаем горизонтальную диаграмму
bars = plt.barh(store_sales['store_name'], store_sales['total_value'], color='skyblue', height=0.7)

#Добавляем значения на диаграмму
for bar, value, percent in zip(bars, store_sales['total_value'], store_sales['revenue_percent']):
    width = bar.get_width()
    plt.text(width + (total_revenue * 0.005), 
             bar.get_y() + bar.get_height()/2, 
             f"{width:,.0f} руб.\n({percent}%)", 
             va='center', 
             fontsize=9)

plt.xlabel('Сумма выручки, руб.')
plt.ylabel('Магазин')
plt.title('Рейтинг магазинов по выручке за 2025 год')
plt.grid(axis='x', linestyle='--', alpha=0.7)
plt.xlim(0, total_revenue * 1.15)
plt.tight_layout()
plt.show()
```

**Что делает код:**

Загрузка данных:

- df_sales — транзакции продаж (объёмы, суммы, даты);

- df_stores — справочник магазинов (ID, названия).

Подготовка дат:

- преобразует столбец sales_time в формат datetime;

- фильтрует записи только за 2025 год.

- Агрегация по магазинам:

- группирует продажи по store_id;

- суммирует:

   - total_qty — количество проданных товаров;

   - total_value — выручку в рублях.

Связывание с названиями магазинов:

- присоединяет колонку store_name через объединение с df_stores.

Расчёт долей:

- вычисляет общую выручку за период (total_revenue);

- определяет долю каждого магазина в общей выручке (revenue_percent).

Сортировка и форматирование:

- упорядочивает магазины по возрастанию выручки (ascending=True);

- формирует итоговую таблицу с понятными заголовками.

Визуализация:

- строит горизонтальную линейчатую диаграмму (barh);

  для каждого магазина отображает:

  - длину столбца — сумма выручки;

  - подпись — выручка в рублях и доля в %;

  - настраивает сетку, подписи осей и заголовок.

**Как читать результат:**

Таблица:

- строки — магазины в порядке возрастания выручки;

колонки:

 - Название магазина — наименование точки;

 - Количество проданных товаров — объём продаж в штуках;

 - Сумма выручки — итоговая выручка в рублях;

 - Доля от общей выручки, % — вклад магазина в совокупный доход сети.

Диаграмма:

- ось Y — названия магазинов (снизу вверх: от минимальной к максимальной выручке);

- ось X — сумма выручки в рублях;

- длина столбца — величина выручки;

- подпись у конца столбца — точная сумма и процент от общего объёма.

Примеры интерпретации:

- Длинные столбцы справа — магазины‑лидеры по выручке. Стоит изучить их практики (ассортимент, сервис, маркетинг) для тиражирования.

- Короткие столбцы слева — точки с низкой выручкой. Требуется анализ:

  - локация и проходимость;

  - ассортимент и цены;

  - работа персонала;

  - конкуренция в районе.

Высокая доля нескольких магазинов — концентрация выручки в отдельных точках. Риск: зависимость сети от их работы.

Равномерное распределение долей — сбалансированная сеть, где каждый магазин вносит сопоставимый вклад.

Резкий разрыв между лидерами и аутсайдерами — возможные проблемы с выравниванием условий работы точек (закупки, логистика, маркетинг).

**Ключевые выводы:**

Топ‑3 магазина по выручке — основные драйверы дохода сети.

Магазины в нижней трети рейтинга — кандидаты на оптимизацию или закрытие.

Доля лидера в общей выручке показывает степень концентрации продаж.

Соотношение количества товаров и суммы выручки выявляет точки с высоким средним чеком.
