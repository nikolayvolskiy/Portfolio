В этом разделе собраны самые распространенные аналитические кейсы:

1. Ранжиррование по выручке.
2. Анализ динамики выручки за определенный период.
3. Анализ показателя Retention и LTV.
4. RFM сегментация клиентов.

Для решение задач были использованы следующие инструменты:

1. Python (через Jupyter Notebook).
2. SQL (через DBeaver). Запросы написаны на двух диалектах - Postgre и SQLite.
