В этом разделе собраны  ad hoc задачи, с которыми я сталкивался в своей работе. Каждая задача имеет детальное описание ее практического смысла и интерпретации данных. Результаты исследований не представлены из соображений конфидециальности данных. 


1. Ранжирование по выручке категорий, точек продаж и сотрудников.

2. Анализ динамики выручки за определенный период.

3. Анализ показателя Retention и LTV.

4. RFM сегментация клиентов.

5. Статистическое исследование расхождений средних чеков у разных уровней программы лояльности.


Для решение задач были использованы следующие инструменты:


1. Python (через Jupyter Notebook).

2. SQL (через DBeaver). Запросы написаны на Postgre.
