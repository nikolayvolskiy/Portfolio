## Описание задачи: проверка статистической значимости расхождений среднего чека у клиентов разных программ лояльности


**Цель анализа:**


Установить, есть ли статистически значимые различия в среднем чеке между клиентами, участвующими в разных программах лояльности (на примере уровней Gold и Silver).


**Исходные данные:**

CSV файл с транзакционными данными (bonus_plus_transaction_data.csv).

Фильтрация: только транзакции типа «покупка» (transaction_type == 'покупка').

Группы для сравнения:

 - Клиенты уровня Gold (card_level == 'gold').  

 - Клиенты уровня Silver (card_level == 'silver').  

 - Анализируемый показатель: сумма покупки в рублях (amount_rub).  


**Методика анализа:**


**Анализ проводится в два этапа:**


1.	Проверка равенства дисперсий (тест Левене)

   	Цель: определить, можно ли применять классический t тест Стьюдента (при равных дисперсиях) или нужно использовать тест Уэлча (при неравных дисперсиях).

  	Гипотезы:

    - H₀: дисперсии равны.  

    - H₁: дисперсии различаются.  

    Критерий принятия решения:

    - Если p value < 0,05 → дисперсии значимо различаются → переходим к тесту Уэлча.  

    - Если p value ≥ 0,05 → нет оснований отвергать равенство дисперсий → можно использовать t тест Стьюдента.  

2.	Сравнение средних значений (тест Уэлча)

    Цель: проверить, различаются ли средние чеки между группами Gold и Silver.

    Гипотезы:

    - H₀: средние чеки равны.  

    - H₁: средние чеки различаются.  

    Критерий принятия решения:

    - Если p value < 0,05 → различия статистически значимы.  

    - Если p value ≥ 0,05 → статистически значимых различий не обнаружено.  


**Выходные данные и интерпретация:**


Программа выводит:


1. Результаты теста Левене:

   - p value и вывод о равенстве дисперсий.


2. Результаты сравнения средних (тест Уэлча):

   - размеры выборок для Gold и Silver.  

   - Средние чеки для каждой группы (в рублях).  

   - T статистика и p value.  

   - Итоговый вывод о наличии/отсутствии значимых различий.  


Ключевые метрики в выводе:

- Средний чек — среднее значение amount_rub в каждой группе.  

- T статистика — мера различия между средними относительно вариабельности данных.  

- P value — вероятность получить наблюдаемые различия при условии, что H₀ верна.  

- Уровень значимости — порог 0,05 (5%).  


**Итоговый результат:**

На основании анализа можно сделать вывод:

•	Есть ли статистически значимые различия в среднем чеке между клиентами Gold и Silver.  

•	Какие группы тратят больше в среднем.  

•	Насколько надёжны выводы (по величине p value).  


```python

import pandas as pd
from scipy import stats
import numpy as np

#Проверка дисперсий
data = pd.read_csv('C:/Python/bonus_plus_transaction data.csv', sep=';', encoding='cp1251', low_memory=False)
purchases = data[data['transaction_type'] == 'покупка']

avg_check_gold = purchases[purchases['card_level'] == 'gold']['amount_rub']
avg_check_silver = purchases[purchases['card_level'] == 'silver']['amount_rub']

levene_stat, levene_p = stats.levene(avg_check_gold, avg_check_silver)

print("Результат проверки дисперсий:")
print(f"P-value теста Левене: {levene_p:.6f}")
if levene_p < 0.05:
    print("Дисперсии значимо различаются → используем тест Уэлча")
else:
    print("Нет оснований отвергнуть равенство дисперсий → можно использовать классический t-тест")
print()

```


В нашем случае дисперсии неравны, поэтому для проверки статистической значимости берем тест Уэлча

```python

#Сравнение средних с неравными дисперсиями (тест Уэлча)
t_stat, p_value = stats.ttest_ind(avg_check_gold, avg_check_silver, equal_var=False)

print("Сравнение средних чеков (тест Уэлча):")
print(f"Размер выборки Gold: {len(avg_check_gold)}")
print(f"Размер выборки Silver: {len(avg_check_silver)}")
print(f"Средний чек Gold: {avg_check_gold.mean():.2f} руб")
print(f"Средний чек Silver: {avg_check_silver.mean():.2f} руб")
print(f"T-статистика: {t_stat:.4f}")
print(f"P-value: {p_value:.6f}")

if p_value < 0.05:
    print("Средние чеки статистически значимо различаются")
else:
    print("Статистически значимых различий не обнаружено")
```
