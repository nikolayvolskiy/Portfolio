**Задача:** создание комплексной аналитической платформы для мониторинга ключевых метрик ритейл компании.


**Инструменты:** Power BI Desktop на удаленном сервере.


**Ключевые этапы:**

1) Загрузка данных в таблицы

2) Трансформация данных в PowerQuery

3) Создание технических таблиц

4) Форматирование данных

5) Настройка связей между таблицами

6) Создание DAX мер

7) Оформление витрин данных


В папке со скриншотами представлены скриншоты всех страниц со всех дашбордов. В файле **Навигация** детальное описание каждого дашборда и его аналитические возможности.


**Результат:** Единая платформа для анализа эффективности работы компании в любом разрезе. Сокращение времени формирования рутинной отчетности до нескольких минут. Улучшения качества принятия решений за счет доступной информация по ключевым показателям.


*Важно: дашборды на скриншотах подключены к тестовому набору данных, сгенерированному случайным образом с заданной структурой, поэтому некоторые показатели могут слегка отличаться от привычных для данной сферы. Реальные данные не представлены из соображений конфидециальности.*


*Так же прошу учесть, что на разных скриншотах могут быть установлены разные фильтры в панели управления, поэтому некоторая информация может показаться несвязанной. Это нормально для демонстрации.*
